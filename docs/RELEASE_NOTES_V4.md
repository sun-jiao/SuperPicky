## 🦜 慧眼选鸟 SuperPicky V4.0.0

> **重大更新**：新增 AI 鸟类识别功能！

---

### 🆕 V4.0.0 新功能

#### 🐦 鸟类识别 (BirdID)
- **AI 自动识别鸟类物种**，支持 11,000+ 鸟类物种
- 智能裁剪鸟类区域，提高识别准确率
- 识别结果自动写入照片 IPTC 元数据（关键词、标题、描述）
- **eBird 集成**：基于 GPS 位置的本地鸟类过滤，提高区域性识别准确率
- 支持国家/地区手动选择

#### 📷 Lightroom 插件
- 在 Lightroom 中直接识别鸟类
- 自动写入 IPTC 关键词和鸟种描述
- 支持批量处理
- 安装程序自动安装插件到 Lightroom

#### 🗂️ 鸟种分类目录
- 按鸟种自动创建子目录
- RAW+JPEG 配对文件同时移动
- 连拍检测支持鸟种子目录

#### 🖥️ 系统托盘支持
- 新增系统托盘图标
- 支持后台运行模式
- 隐藏 Dock 图标选项

#### 🔄 自动更新检测
- 启动时自动检查新版本
- 手动检查更新按钮

---

### 🔧 V3.9.5 ~ V4.0.0 修复与优化

- 修复 ExifTool 路径在打包应用中的问题
- 修复中文字符编码问题
- 修复 JSON NoneType 错误（连拍检测）
- 修复 Windows 兼容性和 ExifTool 路径解析
- 修复临时 JPEG 清理逻辑误删原始 JPEG
- 独立进程 API 服务器架构
- Reset 时自动清理临时转换的 JPEG
- 安装程序支持多个 Lightroom 版本

---

### 💻 系统要求

- macOS 11.0 或更高版本
- Apple Silicon (M1/M2/M3/M4) 或 Intel 处理器
- 约 2GB 可用磁盘空间

---

### 📥 安装说明

1. 下载 DMG 文件
2. 双击打开 DMG
3. 运行 `SuperPicky_v4.0.0_Installer.pkg` 安装程序
4. 安装程序会自动安装主应用和 Lightroom 插件

---

### ⚠️ 首次使用注意

- 首次运行需要 10-30 秒加载 AI 模型
- 使用 Lightroom 插件前需先启动主应用
- Lightroom 插件需手动启用：文件 → 增效工具管理器 → 找到「慧眼选鸟」→ 点击「启用」
